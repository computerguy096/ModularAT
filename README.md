# ModularAT

A fully 3D printable computer case for Baby AT shaped motherboards, with new features the originals never had.

- modular design, can be assembled in a much smaller version
- compatible with all Baby AT motherboards (up to 330mm deep in the full configuration)
- customizable rear I/O area: supports motherboards with AT + PS/2 mouse combo, dual PS/2, etc
- 120mm intake fan: silent operation with plenty of airflow
- three hard drive mount points
- two 5.25 inch (half height, full length) drive bays
- one 3.5 inch drive bay for floppy drives
- ATX power supply support
- improved cable management, with dedicated space behind the motherboard tray for cables
- does not use those pesky plastic pegs!

It is currently under development so it is possible some parts might not be finalized, print at your own risk.
